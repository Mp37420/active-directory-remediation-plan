# Active Directory Security Remediation Plan

## Overview
Performed a security assessment of an Active Directory environment and developed a remediation plan addressing critical vulnerabilities impacting security, compliance, and operations.

## Key Vulnerabilities
- Outdated Operating Systems
- GPP Password Exposure
- IPv6 & LLMNR/NBT-NS Poisoning
- Kerberoasting & AS-REP Roasting
- SMBv1 Usage
- Sensitive Data Exposure

## Remediation Strategy
- System upgrades and patching
- Secure credential management (gMSA, LAPS)
- Disabling insecure protocols
- Strengthening access controls and monitoring

## Impact
- Reduced attack surface
- Improved compliance posture
- Enhanced protection of sensitive data
